# Shared Agent Instructions

Project: **fe**
Stack: **Laravel**
Variant: **Web**
Scaffold Version: **2.0.0**

## Operating Standard
- Start at `.agent/INDEX.md`.
- Read the active spec in `.agent/specs/active/` before implementing feature work.
- Stay inside the requested task scope.
- Read only the files needed for the task.
- Record progress and verification in `.agent/tasks/current-task.md`.
- Update long-lived docs only when facts or conventions actually changed.

## Variant Priorities
- Prioritize routing, rendering, accessibility, and state boundaries.
- Keep UI components composable and explicit.
- Document user flows, visual system, and browser testing.
