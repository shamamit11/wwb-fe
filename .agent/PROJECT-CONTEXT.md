# Project Context

## Product Summary
fe is a frontend or browser-first application built on Laravel.

## Stakeholders
1. End users who rely on correctness and clear behavior.
2. Maintainers who need explicit module boundaries and commands.
3. AI agents that should work with narrow context, strong task discipline, and explicit specs.

## Variant Priorities
- Prioritize routing, rendering, accessibility, and state boundaries.
- Keep UI components composable and explicit.
- Document user flows, visual system, and browser testing.
